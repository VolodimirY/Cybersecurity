## 🇪🇸 Introducción a Mocha

**Mocha** es un framework de testing para **JavaScript** que se ejecuta sobre **Node.js**. Su objetivo es facilitar la creación y ejecución de pruebas automatizadas para asegurar que el código funcione correctamente. Es una herramienta muy popular por su **sencillez**, **flexibilidad** y **compatibilidad** con distintas librerías de aserciones, como **Chai**.

Con Mocha puedes escribir tests que describan el comportamiento esperado de tus funciones o módulos, y luego ejecutarlos automáticamente para detectar errores antes de que tu aplicación llegue a producción. Gracias a su sintaxis clara y su integración con herramientas como **Chai**, Mocha permite crear pruebas legibles y fáciles de mantener.

### 📘 En esta guía aprenderás a:
- Instalar y configurar Mocha en un proyecto **Node.js**.  
- Escribir tus primeros **tests**.  
- Utilizar **Chai** para hacer comprobaciones (aserciones).  
- Ejecutar y entender los resultados de las pruebas.  

🎯 **Objetivo:** Al finalizar, podrás crear tus propios tests unitarios y comprender el flujo básico de pruebas en JavaScript.

## 🇬🇧 Introduction to Mocha

**Mocha** is a **JavaScript testing framework** that runs on **Node.js**. Its main goal is to make it easier to create and execute automated tests to ensure your code works correctly. Mocha is widely used because of its **simplicity**, **flexibility**, and **compatibility** with various assertion libraries such as **Chai**.

With Mocha, you can write tests that describe the expected behavior of your functions or modules and run them automatically to catch errors before your application goes to production. Thanks to its clear syntax and integration with tools like **Chai**, Mocha helps developers create readable and maintainable test suites.

### 📗 In this guide, you will learn how to:
- Install and set up **Mocha** in a **Node.js** project.  
- Write your first **tests**.  
- Use **Chai** for assertions.  
- Run and interpret your test results.  

🎯 **Goal:** By the end, you’ll be able to create your own unit tests and understand the basic testing workflow in JavaScript.
